Simple Console Printer Application

1.Open Eclipse IDE, Open Printer folder and under Printer\src\Printer there is Printer.java. 
2.Press Run - Ctrl + F11
3.Program will prompt "Please enter H, L, X, Y or Z"
4.After entering either H, L, X, Y or Z 
5.Program will prompt "Please enter Odd Numbers from 3 to 21"
6.After entering a odd number from 3 to 21. It will print the asterisk character input from (3)
7.Program will prompt again if
(3) is not H, L, X, Y or Z AND 
(5) is less than 3 and more than 21 and its not an ODD number.
